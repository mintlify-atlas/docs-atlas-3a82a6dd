## What IRB does

IRB operates as a **Read-Eval-Print Loop** (REPL):

- **Reads** input character by character with syntax-aware editing
- **Evaluates** code once it forms a syntactically complete expression
- **Prints** the result with syntax-highlighted, inspected output

```shell
$ irb
irb(main):001> 1 + 2
=> 3
irb(main):002* class Greeter
irb(main):003*   def hello(name)
irb(main):004*     "Hello, #{name}!"
irb(main):005*   end
irb(main):006> end
=> :hello
irb(main):007> Greeter.new.hello("Ruby")
=> "Hello, Ruby!"
```

<Steps>
  <Step title="Start IRB">
    Run `irb` from your terminal. IRB is included with Ruby — no installation needed.
  </Step>
  <Step title="Evaluate expressions">
    Type any Ruby expression and press Enter. IRB evaluates it and prints the result.
  </Step>
  <Step title="Explore your code">
    Use `ls`, `show_source`, and `show_doc` to inspect objects and methods on the fly.
  </Step>
  <Step title="Customize your session">
    Add configuration to `~/.irbrc` to set your preferred prompt, history size, and more.
  </Step>
</Steps>
